#Learn git
## 주요 단어

- Working directory:사용자가 작업한 파일들이 저장되는 곳

- Local repository:파일이 변경된 이력이 저장되는 곳

- Remote repositoy:로컬저장소의 파일을 안전한 곳에 저장하기 위한 원경저장소

##명령어

git init

git add

git commit
